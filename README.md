# Dev front end usuario de React e Sass

#### Testes

Aqui vou realizar alguns testes de componente com a [Biblioteca de testes react](https://testing-library.com/docs/react-testing-library/intro/)

# Tecnologias que utilizo

- React.js
- Styled components
- Scss
- Eslint
- Jest

# Patterns

Padrões de projeto que eu utilizo geralmente.

O css é padronizado com [BEM](https://desenvolvimentoparaweb.com/css/bem/).

O Jsx é padronizado pelo eslint com configuração [standard](https://standardjs.com/).

O Styled components possui um estilo global no caminho ```src/components/``` e as variaveis
e estilos reutilizaveis no caminho ```src/UI/```.
