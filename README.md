# Css repo training

Repositorio para treinar SCSS CSS Styled-components e React.js

# Tecnologias

- React.js
- Styled components
- Scss
- Eslint

# Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/cesar-abj/css.git

# Acesse o diretório
cd dir_do_projeto

# Instale as dependencias
yarn install

# Inicie o projeto
yarn start
```
